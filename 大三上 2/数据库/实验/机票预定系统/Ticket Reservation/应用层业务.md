## 1. 认证与安全

- PBKDF2 + 每用户 Salt（60 万迭代），旧库自动补列，首次登录自动从明文升级为哈希：见 [business_operations/service.py](business_operations/service.py) 中 `_hash_password`、`_ensure_security_columns`、`authenticate`。
```python
# business_operations/service.py 片段
class TicketingService:
    FAILED_LIMIT = 5
    LOCK_MINUTES = 1
    @staticmethod
    def _hash_password(password: str, salt: Optional[str] = None) -> Tuple[str, str]:
        salt_bytes = base64.b64decode(salt) if salt else os.urandom(16)
        dk = hashlib.pbkdf2_hmac("sha256", password.encode("utf-8"), salt_bytes, 600000)
        return base64.b64encode(salt_bytes).decode(), base64.b64encode(dk).decode()
```
- 登录失败 5 次锁定 1 分钟（失败计数写回；锁定过期自动解锁）：见 [business_operations/service.py](business_operations/service.py) 的 `authenticate`。
- 修改密码校验当前密码（含 Salt），重置失败计数/锁定标记：`change_password` 函数。[business_operations/service.py](business_operations/service.py)
```python
# business_operations/service.py 片段
```

- 默认管理员自动创建（admin/admin123，哈希存储）：`_ensure_admin()`。[business_operations/service.py](business_operations/service.py)
