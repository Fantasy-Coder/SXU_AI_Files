田学长关于试卷使用的几点提示：
1. 空白试卷以学科命名，前加下划线'_'是指对应科目试卷答案
2. 配套讲解链接为B站【晓月波涛的个人空间-哔哩哔哩】 https://b23.tv/75M7OLR
3. 试卷仅限计算机相关专业本科生期末复习使用，考研等其他更高水平考试至少应当在当年7月前掌握对应科目试卷内所有知识点
4. 此均系本人一人完成试卷出题、答案书写、试卷讲解工作，不盈利且由于是服务同学们期末复习，时间较为紧迫，疏漏之处在所难免，烦请包涵或指出
5. 操作系统、计算机网络两个科目试卷当中部分题目来源王道。
6. 操作系统当中对于PV操作部分的题目较为粗浅，是由于当学期期中考试重点考了PV操作，当学期期末考试对PV操作考察较为简单，若有更高学习需求的同学请继续深耕王道复习书
7. 希望此试卷可以为同学们的期末复习起到抛砖引玉的作用，感谢各位厚爱
8. 期待各位一键三连+关注，后续有更多精彩更新！
